How to Play
A Conqueror of the Seven Hand Game — Alchemy Guide

A Conqueror of the Seven-Hand Game is an evolution of Rock-Paper-Scissors with 7 elements that defeat one another. Each element beats 3 other elements and loses to 3 other elements.

Game Flow
1
Leader creates lobby and chooses game mode
2
Other players join via Lobby ID
3
Each round, all players choose 1 element simultaneously
4
Server calculates results — winner gets points / HP preserved
✓
Win condition depends on mode — see Game Modes tab
⏱
Limited Timer
Each round has a timer. If you don't choose, your last element is used (or random if you haven't chosen yet).
🤖
Auto Bots
If playing solo, bots automatically fill empty slots when game starts.
🤝
Draw
If both players choose the same element, the round is a draw. No points/HP change.
👁
Spectator
Players whose HP runs out (lives mode) enter spectator mode — still watch and chat.

Click an element to see full details — what it beats and what beats it.
